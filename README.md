# advent-of-code-js

## Clear, efficient and didactic solutions 

The code is written in the most basic vanilla JavaScript, trying to be very accessible for everybody.

For some puzzles, **more than one algorithm** is published, allowing one to compare the differences in logic and efficiency (which is huge sometimes).

I started at the 2015 event. 

The plan is to publish solutions for all  Advent Of Code puzzles.
